Rust中的Vector(向量)

Awsome
Rust
发布日期:   2025-02-22
更新日期:   2025-02-22
文章字数:   1.5k
阅读时长:   6 分

Rust 中的 Vector (向量) 是一种动态存储的数组，允许在运行期增加或删除元素。

1. 创建 Vector

Rust 中使用 Vec<T> 来实现 Vector。例如：

fn main() {
    let v: Vec<i32> = Vec::new(); // 创建一个空的 Vector
    let v = vec![1, 2, 3, 4, 5]; // 使用 vec! 快速创建
}

1.1 添加元素

可以使用 push 方法向 Vector 中添加元素：

fn main() {
    let mut v = Vec::new();
    v.push(10);
    v.push(20);
    v.push(30);
    println!("{:?}", v); // [10, 20, 30]
}

2. 访问 Vector 元素

2.1 通过下标访问

fn main() {
    let v = vec![10, 20, 30, 40];
    let third = &v[2];
    println!("The third element is {}", third); // 30
}

注意：如果跨界访问，会出现运行时错误：

fn main() {
    let v = vec![1, 2, 3];
    let invalid = &v[10]; // 跨界会 panic!
}

2.2 通过 get 方法访问

fn main() {
    let v = vec![10, 20, 30, 40];
    match v.get(2) {
        Some(third) => println!("The third element is {}", third),
        None => println!("No third element."),
    }
}

3. 遍历 Vector

3.1 使用 for 循环

fn main() {
    let v = vec![10, 20, 30];
    for i in &v {
        println!("{}", i);
    }
}

3.2 可变遍历

fn main() {
    let mut v = vec![10, 20, 30];
    for i in &mut v {
        *i += 1;
    }
    println!("{:?}", v); // [11, 21, 31]
}

4. 添加不同类型元素

Rust 中 Vector 一般必须包含相同类型元素，如果需要添加不同类型的元素，可以使用 enum：

enum Data {
    Int(i32),
    Float(f64),
    Text(String),
}

fn main() {
    let v = vec![
        Data::Int(42),
        Data::Float(3.14),
        Data::Text(String::from("Hello")),
    ];
}

5. Vector 存在期间问题

注意：不能同时存在字符串的取得和向量的修改！

fn main() {
    let mut v = vec![String::from("Hello")];
    let first = &v[0]; // 元素的存在期存在问题
    v.push(String::from("World")); // 此处会报错
    println!("{}", first);
}

解决方案：先处理参考后进行修改

fn main() {
    let mut v = vec![String::from("Hello")];
    {
        let first = &v[0];
        println!("{}", first);
    } // 此处保证元素在添加前已被释放
    v.push(String::from("World"));
}

6.完整代码演示

use std::fmt;

fn main() {
    // 创建一个向量并使用不可变引用
    let v = create_vector(); 
    let v_ref = &v; // 获取 v 的不可变引用

    // 访问向量并打印第二个元素
    access_vector(v_ref);

    // 使用 for 循环迭代打印向量中的每个元素
    iterating_by_for(v_ref);

    // 创建一个可变向量
    let mut v2 = create_vector(); 
    let v2_ref = &mut v2; // 获取 v2 的可变引用

    // 修改向量中的元素并访问
    iterating_and_modif(v2_ref);

    println!("Access second element again");

    // 再次访问修改后的向量并打印第二个元素
    access_vector(v2_ref);

    println!("crate_differnce_type_vec");

    // 创建并打印不同类型的向量
    let v3 = crate_differnce_type_vec();

    // 演示向量的修改与访问的借用规则
    modif_and_access();
}

// 创建一个包含整数的向量并返回
fn create_vector() -> Vec<i32> {
    //使用vec! 快速创建
    //let v = vec![1,2,3];

    // vec! 是一个快速创建宏，但这里使用 Vec::new() 创建空向量
    let mut v = Vec::new();
    v.push(1); // 向量添加元素
    v.push(2);
    v.push(3);
    return v; // 返回创建的向量
}

// 通过不可变引用访问向量并打印第二个元素
fn access_vector(v: &Vec<i32>) {
    // 使用下标访问向量元素，如果越界会 panic!
    let second_element = &v[1];
    println!("the second element is {}", second_element);

    // 使用 get() 方法安全访问元素，如果下标越界会返回 None
    let second_element = &v.get(1);
    match second_element {
        Some(value) => println!("the second element is {}", value),
        None => println!("not exist!"), // 如果元素不存在，打印 "not exist!"
    }
}

// 通过 for 循环迭代不可变引用的向量并打印每个元素
fn iterating_by_for(v: &Vec<i32>) {
    for i in v {
        println!("iterating_by_for :{}", i); // 打印每个元素
    }
}

// 通过可变引用修改向量中的元素
fn iterating_and_modif(v: &mut Vec<i32>) {
    // 使用 iter_mut() 获取可变引用，并修改每个元素
    for i in v.iter_mut() {
        *i += 50; // 将每个元素加 50
    }

    // 在修改后通过不可变引用再次访问向量
    access_vector(v); // 注意，Rust 允许在某些情况下将可变引用转换为不可变引用
}

// 定义一个结构体 Product
struct Product {
    id: i32,
    name: String,
}

// 为 Product 实现 Debug trait 以便打印
impl fmt::Debug for Product {
    fn fmt(&self, f: &mut fmt::Formatter) -> fmt::Result {
        write!(f, "Product {{ id: {}, name: {} }}", self.id, self.name)
    }
}

// 定义一个枚举 Data，包含不同的数据类型
enum Data {
    Int(i32),
    Float(f64),
    Text(String),
    Product(Product),
}

// 为 Data 实现 Debug trait 以便打印
impl fmt::Debug for Data {
    fn fmt(&self, f: &mut fmt::Formatter) -> fmt::Result {
        match self {
            Data::Int(i) => write!(f, "Int({})", i), // 打印整数
            Data::Float(fl) => write!(f, "Float({})", fl), // 打印浮点数
            Data::Text(txt) => write!(f, "Text({})", txt), // 打印字符串
            Data::Product(p) => write!(f, "Product({:?})", p), // 打印 Product
        }
    }
}

// 创建一个包含不同类型数据的向量并返回
fn crate_differnce_type_vec() -> Vec<Data> {
    let v = vec![
        Data::Int(32), // 整数
        Data::Float(64.0), // 浮点数
        Data::Text(String::from("Text")), // 字符串
        Data::Product(Product { // Product 结构体
            id: 1,
            name: String::from("product!"),
        }),
    ];

    // 打印每个元素
    for i in &v {
        println!("{:?}", i); // 使用 {:?} 打印 Debug trait 实现
    }

    return v; // 返回向量
}

// 演示向量的修改与访问时的借用规则
fn modif_and_access() {
    let mut v = vec![String::from("hello")];

    // 错误示例：不能同时进行元素访问和修改
    // 以下代码会导致借用冲突错误
    // let first = &v[0]; 
    // v.push(String::from("world")); // 这里进行修改操作
    // println!("{}", first);

    { // 使用一个新的作用域来隔离借用和修改
        let first = &v[0]; // 获取不可变引用
        println!("{}", first); // 打印第一个元素
    } // 在此作用域结束后，借用解除

    v.push(String::from("world")); // 修改向量（添加新元素）
}

7. 总结

  • Rust 中的 Vector 使用 Vec<T> 实现。
  • 可以通过 vec![] 创建向量，或使用 Vec::new() 并使用 push 添加元素。
  • 访问元素时，可以使用下标或 get()，后者可避免跨界错误。
  • 循环 Vector 可以使用 for ，如果需要修改元素，需要使用 &mut。
  • 不同类型元素可以使用 enum 实现。
  • Vector 保持透明的所有权限，避免漏洞，需注意存在期问题。
